MEMs - TTE

This repository contains all files and code used in the simulation studies that evaluate the multisource exchangeability models for incorporating historical studies with a time-to-event endpoint.

Simulation code are included for the simulations studies described in the manuscript "Bayesian modeling with time-to-event data using multisource exchangeability models".

A case study is also inclueded and the data is from public use dataset "A Multicenter, Adaptive, Randomized, Blinded Controlled Trial of the Safety and Efficacy of Investigtational Therapeutics for Hospitalized Patients with COVID-19 (TICO)" which can be accessed upon request.
